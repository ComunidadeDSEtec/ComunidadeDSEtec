## Objetivo da comunidade:
<ul>
<li>Unificar os Desenvolvedores de sistemas das Etecs;</li>
<li>Manter todos informados sobre possíveis vagas, cursos e até tendências do mercado;</li>
<li>Facilitar a inclusão e acolhimento dos calouros, principalmente os que são leigos;</li>
<li>Criar uma rede de networking, promover interação social com profissionais similares a você;</li>
</ul>

## Para quem é a comunidade:
<ul>
<li>Leigos sobre Ti, que procurem imersividade na área, de forma mais simplória;</li>
<li>Desesperados por oportunidades no mercado de trabalho. OBS: oportunidades de cargo baixo, analista, desenvolvedor júnior, pleno e estagiário;</li>
<li>Especializações, quem estiver procurando cursos, dicas práticas, projetos, métodos e guia para carreira;</li>
<li>Migrantes, aqui também é para quem caiu de paraquedas e mudou de área.</li>
</ul>
